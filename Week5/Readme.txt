Please view the write up in Google Docs using the url on the front page of the report.  The code and progressive views are best viewed inside Jupyter Notebook, but the data download is over three terabytes.  

The visualization work progressed from data cleaning, to frame renderings using Img, PIL, matplotlib inside python-base Jupyter notebooks.

The second improvements cycle used the frames to create perspective videos and individual subject videos for all 1,247 scans.

The third improvement cycle used Plot.ly and Dash to build a completely custom dashboard application.  This application allows the user the ability to pull up any subject's scan and pan through all perspective data using sliders.
